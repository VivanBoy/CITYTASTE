# Détails d’un lieu — Comprendre la fiche détaillée

## À quoi sert la fiche détaillée ?

La fiche détaillée sert à afficher plus d’informations sur un lieu sélectionné dans les résultats. Elle aide l’utilisateur à mieux comprendre l’endroit avant de décider s’il veut le visiter, le comparer ou consulter son site web.

La fiche détaillée est particulièrement utile quand l’utilisateur veut aller au-delà d’une simple carte de résultat.

---

## Quelles informations peut-on trouver dans une fiche détaillée ?

Selon les données disponibles, une fiche détaillée peut contenir :

- le nom du lieu
- le type d’établissement
- la cuisine pour les restaurants
- l’adresse
- les horaires
- le site web
- certaines informations d’accessibilité
- la note et le nombre d’avis lorsqu’ils existent
- une image si elle est disponible
- un texte descriptif ou enrichi

Toutes les fiches n’ont pas exactement le même niveau de détail.

---

## Pourquoi certaines informations manquent dans la fiche ?

Il est possible qu’un lieu n’ait pas :
- d’horaire précis
- de site web
- de photo
- de note
- d’information complète sur l’accessibilité

Cela dépend de la qualité des données disponibles. La fiche ne doit pas inventer ce qui manque. Elle doit afficher clairement l’information lorsqu’elle existe, et rester sobre lorsqu’elle est absente.

---

## Comment ouvrir la fiche d’un lieu ?

En général, l’utilisateur ouvre la fiche détaillée en cliquant sur un résultat ou sur une carte associée à ce lieu dans l’interface.

Le site peut ensuite afficher une vue plus complète contenant les informations connues sur cet établissement.

---

## La fiche détaillée suffit-elle pour prendre une décision finale ?

La fiche détaillée aide beaucoup, mais elle ne remplace pas toujours une vérification externe. Par exemple, si l’utilisateur veut confirmer :
- les horaires exacts
- les disponibilités
- une réservation
- des conditions particulières

il peut être nécessaire de consulter le site officiel du lieu ou un service externe.

---

## Pourquoi deux fiches détaillées n’ont-elles pas la même richesse ?

Parce que les lieux n’ont pas tous la même quantité d’informations disponibles dans les données sources et les enrichissements associés.

Un lieu peut avoir :
- une note et une photo
- une adresse et un site web
- une description enrichie

Un autre peut n’avoir qu’un nom et une adresse partielle. Cela ne veut pas dire qu’il est moins bon, seulement qu’il est moins documenté dans la base.

---

## Formulations fréquentes

L’utilisateur peut poser des questions comme :

- Comment voir les détails d’un lieu ?
- Que contient la fiche détaillée ?
- Pourquoi je ne vois pas toutes les informations ?
- Est-ce que je peux voir l’adresse complète ?
- Est-ce que la fiche montre les horaires ?
- Comment ouvrir la page d’un lieu ?
- Pourquoi la fiche d’un lieu est plus complète qu’une autre ?
