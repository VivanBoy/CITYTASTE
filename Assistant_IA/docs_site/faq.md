# FAQ générale — CityTaste

## Qu’est-ce que CityTaste ?

CityTaste est une application web de recommandation qui aide les utilisateurs à découvrir des **restaurants** et des **hébergements** à Ottawa selon leurs préférences. Le site permet d’explorer les lieux disponibles, de filtrer les résultats et d’ouvrir une fiche détaillée lorsqu’une information plus complète est disponible.

CityTaste ne remplace pas un service de réservation. Son rôle principal est d’aider l’utilisateur à **trouver**, **comparer** et **mieux comprendre** les lieux proposés à partir des données disponibles.

---

## Quels types de lieux peut-on trouver sur le site ?

CityTaste couvre principalement deux grandes catégories :

- les **restaurants**
- les **hébergements**

Selon les données disponibles, la catégorie hébergement peut inclure plusieurs types de lieux comme :
- hôtel
- motel
- auberge
- guest house
- hostel

Le site peut aussi afficher différentes cuisines pour les restaurants, par exemple italienne, indienne, chinoise, vietnamienne, africaine ou d’autres styles selon les données présentes.

---

## Comment utiliser CityTaste ?

Le fonctionnement général est simple :

1. l’utilisateur indique ce qu’il cherche
2. il peut appliquer des filtres selon ses préférences
3. le site affiche une liste de résultats pertinents
4. l’utilisateur peut consulter les détails d’un lieu si une fiche détaillée est disponible

L’objectif est de rendre la recherche plus rapide et plus claire, même lorsqu’il y a plusieurs options possibles.

---

## Est-ce que CityTaste effectue des réservations ?

Non. CityTaste n’effectue pas de réservation à la place de l’utilisateur.

Le site sert à :
- découvrir des lieux
- consulter certaines informations utiles
- comparer plusieurs options
- accéder, lorsque disponible, au site web du lieu ou à ses informations principales

Si l’utilisateur souhaite réserver, il doit généralement passer par le site officiel du lieu ou par un service externe.

---

## Est-ce que toutes les informations sont toujours disponibles ?

Non. Certaines fiches peuvent contenir des informations incomplètes.

Par exemple, il peut arriver qu’un lieu n’ait pas :
- de photo
- de note
- d’horaire détaillé
- de site web
- d’adresse complète

Cela dépend des données disponibles dans les sources utilisées par CityTaste. Quand une information manque, le site essaie de rester transparent plutôt que d’inventer.

---

## Est-ce que les résultats affichés représentent tous les lieux existants à Ottawa ?

Pas nécessairement.

CityTaste s’appuie sur les données disponibles dans son système. Cela signifie que :
- certains lieux peuvent ne pas apparaître
- certaines informations peuvent être absentes ou anciennes
- la couverture dépend de la qualité et de la disponibilité des données

Le site vise à proposer des résultats utiles et crédibles, mais il ne prétend pas représenter l’ensemble absolu de tous les lieux existants.

---

## Le site répond-il aussi aux questions sur son fonctionnement ?

Oui. CityTaste peut aussi aider l’utilisateur à comprendre :
- comment fonctionne la distance
- comment utiliser les filtres
- pourquoi certaines informations sont manquantes
- comment ouvrir la fiche détaillée d’un lieu
- ce que signifie une note ou une photo affichée sur une carte de résultat
